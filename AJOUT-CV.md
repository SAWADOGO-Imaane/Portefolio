# Instructions pour ajouter votre CV

## 📄 Étape à suivre

Pour activer le téléchargement de votre CV, placez votre fichier CV au format PDF dans le dossier du portfolio avec le nom exact suivant :

**Nom du fichier :** `CV_Imaane_SAWADOGO.pdf`

**Emplacement :** Même dossier que `index.html` (racine du portfolio)

```
Portefolio/
├── index.html
├── styles.css
├── script.js
├── CV_Imaane_SAWADOGO.pdf  ← Placez votre CV ici
└── images/
```

## ✅ Une fois le CV ajouté

Le bouton "Télécharger mon CV" dans la section Contact fonctionnera automatiquement et téléchargera votre CV au format PDF quand les visiteurs cliqueront dessus.

## 📝 Note

Si vous souhaitez utiliser un nom de fichier différent, modifiez la ligne 420 dans `index.html` :
```html
<a href="CV_Imaane_SAWADOGO.pdf" download="CV_Imaane_SAWADOGO.pdf" class="btn-download-cv">
```

Remplacez `CV_Imaane_SAWADOGO.pdf` par le nom de votre fichier.
